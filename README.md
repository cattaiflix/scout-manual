# ⚽ Scout Manual

Aplicativo para análise tática de jogos de futebol de campo.

## Funcionalidades
- Registro de informações por tempo (formação, ataque, defesa, saída de pressão, destaques)
- Placar por tempo com tipo de gol (rolando, falta, pênalti, lateral)
- Análise de cobranças de pênalti com mapa de calor do gol
- Ditado por voz em todos os campos de texto
- Geração de relatório em PDF

## Como rodar localmente

```bash
npm install
npm start
```

## Como fazer deploy na Vercel

1. Faça push para o GitHub
2. Acesse vercel.com e importe o repositório
3. Clique em Deploy

Nenhuma configuração extra necessária — a Vercel detecta o Create React App automaticamente.
